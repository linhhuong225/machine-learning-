# Employee Performance Classification Using Machine Learning

## Project Overview

This project applies Machine Learning techniques to classify employee performance based on HR-related attributes.

The goal is to support Human Resource Analytics by identifying patterns that influence employee performance and comparing multiple classification algorithms.

## Business Problem

Employee performance evaluation is an important component of workforce management.

Organizations need data-driven approaches to:

* Identify high-performing employees
* Detect employees requiring additional support
* Improve workforce planning
* Support HR decision-making

This project investigates whether employee performance can be predicted using historical HR data.

## Dataset Information

The dataset contains employee information including:

* Age
* Education Level
* Department
* Job Role
* Job Level
* Employment Type
* Base Salary
* Store Location
* Manager Status

Dataset size:

* 7,500 employee records
* 18 features
